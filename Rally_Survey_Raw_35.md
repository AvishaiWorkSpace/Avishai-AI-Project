This file holds BOTH waves so agents can compute the 26→35 delta.

## KEY DELTAS (26 → 35) — headline
1. SUPPLY rose: "no courts" became #1 blocker (45.7%); multi-app 77% solid. Court availability is now co-equal with (or above) matchmaking.
2. FILL-THE-SPOT intensified: WhatsApp substitute search 54%→66%.
3. LEVEL-TRUST strengthened & broadened: ~74% of open-text; NEW "can't know level without playing" + social-vetting trust signal.
4. MARKETPLACE stays WEAK: money-loss mostly "rare/never/refunded."
5. Sample slightly less newbie-skewed but still active 2-3x/wk core.
