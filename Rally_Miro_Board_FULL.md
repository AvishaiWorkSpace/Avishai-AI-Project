# Rally — לוח Miro מלא (מוכן להדבקה)

> **שימוש:** פתח לוח Miro חדש. לכל Frame למטה — צור frame, העתק את הבלוק, הדבק על הקנבס (Miro יוצר sticky לכל שורה). צבע לפי המקרא.
> **מקרא צבעים:** 🟡 תיקוף · 🟢 מוצר · 🔵 מכירות · 🟣 קהילה · 🟠 עסקי · ⚪ MatchPointer (אפור)

## FRAME 1 — מה זה Rally (סיכום)
```
Rally — פלטפורמת B2B SaaS למועדוני פאדל בישראל
הג'וב: "לשחק עכשיו / למלא מגרשים ריקים"
לקוח משלם: המועדון. מנוע רכישה: אפליקציית השחקנים
זווית: לא הזמנת מגרש (Playtomic), לא דירוג (MatchPointer) — מילוי מגרש
ראש חוף: עיר אחת, שני הצדדים (גוש דן)
שאיפה: עסק רווחי עצמאי (בוטסטראפ), תיקוף לפני השקעה
```

## FRAME 2 — החלטות אסטרטגיות (מהייעוץ)
```
מודל הכנסה: מנוי B2B חודשי למועדונים
תמחור: ₪490 Early Adopter (10 ראשונים) → ₪790 → ₪1,490
שאיפה: עסק רווחי עצמאי, לא גיוס
תקציב: תיקוף קודם, אז להחליט על השקעה
זמן: משרה מלאה (30+ ש'/שבוע)
יעד: מועדון פיילוט משלם תוך 4-6 שבועות
גישה למועדונים: מתחיל מאפס (cold outreach)
נתיב לרווחיות: ~50 מועדונים = ₪32K/חודש
```

## FRAME 3 — ⭐ השוואה: Rally מול MatchPointer (המרכז)

### מפת מיצוב 2x2
צירים: X = ספונטני ↔ מאורגן · Y = ערך לשחקן (סטטוס) ↔ ערך למועדון (תפוסה)
```
MatchPointer — מאורגן × ערך לשחקן (תחרות+דירוג מאומת)
Playtomic — מאורגן × הזמנת מגרש (רפרנס)
RALLY ★ — ספונטני × ערך למועדון (תפוסה) = הרבעון הריק
```

### טבלת השוואה (הדבק כשורות, או בנה טבלה)
```
ממד | MatchPointer | Rally
הג'וב | תעד הישגים + דירוג מאומת | לשחק עכשיו / למלא מגרש
רגש | סטטוס, תחרותיות | ספונטניות, שייכות
תדירות | אירועי (טורניר/ליגה) | יומיומי (פיקאפ)
לקוח משלם | מועדון (תחרויות) | מועדון (תפוסה)
קהל ליבה | שחקן תחרותי | שחקן חברתי-מזדמן
ROI למועדון | בינוני (אירועים) | גבוה (שעות מתות)
זמן אמת | חלש | ליבה
דירוג | ליבה (מאומת) | קל/משני
```

## FRAME 5 — ניקוד משוקלל (3 ריצות רגישות)
```
נוסחה: Score = Σ(משקל × ציון 1-5)
MatchPointer: 3.20 / 3.35 / 3.00 → ממוצע 3.18
Rally נוכחי: 3.35 / 3.35 / 3.35 → ממוצע 3.35
Rally מומלץ: 4.55 / 4.60 / 4.45 → ממוצע 4.53
⚠️ תובנה: בלי בידול = תיקו (3.35) עם מתחרה שכבר מבוסס
✅ עם בידול = פער ברור (4.53 מול 3.18)
```

## FRAME 6 — 7 מהלכי שיפור (🟢)
```
1. מקם מחדש: פיקאפ בזמן אמת, לא leaderboard
2. חתוך סקופ: בלי דירוג ארצי ב-MVP
3. חיה ברבעון הריק: ספונטני × תפוסה
4. דשבורד תפוסה (₪ שחזרו משעות מתות)
5. יעד קהל: השחקן החברתי-מזדמן
6. קואקזיסטנס: כבד/ייבא את הדירוג שלהם — אל תילחם עליו
7. רווי אזור אחד מהר (defensibility)
```
